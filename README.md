# CBT-5C

Thought Record (TR) is a core assessment tool in Cognitive Behavioural Therapy (CBT), playing a crucial role in identifying cognitive distortions and carrying out cognitive restructuring. The absence of large-scale, high-quality, fully-annotated thought record datasets---particularly those covering the complete ``situation--automatic thoughts--cognitive distortions--emotions--alternative thoughts'' chain---is a critical bottleneck hindering the development of AI-assisted CBT. In this paper, we present CBT-5C, the first large-scale structured dataset for Chinese cognitive behavioural therapy research, comprising 7,500 fully-annotated, high-quality samples in a five-column thought record format. CBT-5C extends the existing C2D2 resource by adding two clinically essential fields: emotion labels (across seven categories) and alternative thoughts (generated via a multi-model collaborative pipeline with expert validation). We provide a comprehensive description of the dataset schema, statistical distributions, annotation pipeline, and quality metrics (Cohen's Kappa = 0.992 for emotion labels, 0.664 for alternative thought quality). Fine-tuning experiments on both large-scale (Qwen-2.5-7B) and lightweight (ChatYuan-large-v2) models demonstrate that CBT-5C substantially improves models' cognitive restructuring capabilities.
